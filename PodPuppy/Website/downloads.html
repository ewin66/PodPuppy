Warning: Upgrading from a previous version

All PodPuppy settings and subscriptions should be retained when upgrading, but in case something goes wrong its a good idea to export your subscriptions before upgrading, then if neccesary you can import them again afterwards.

The latest version of PodPuppy is Beta v0.7.0.0 Download it here. If you are unable to run it you may require the Microsoft .NET Framework 2.0 which is a free download from Microsoft.
